# 🚀 React Form Validation + Popup (No Libraries)

Hey there! 👋  
This is a simple React form with live validation and a clean success popup built using **React Hooks** and **Tailwind CSS**. No extra libraries, just pure React goodness.

---

## ✨ Features

✅ Name, Email, Password, Confirm Password fields  
✅ Live validation with instant error messages  
✅ Email format check 🔍  
✅ Password match validation 🔐  
✅ Sweet success popup message 🎉  
✅ Form clears itself after successful submission  

---

## 📦 How to Use

### 1️⃣ Install dependencies

```bash
npm install
````

---

### 2️⃣ Start the dev server

```bash
npm start
```

Open your browser and head to 👉 [http://localhost:3000](http://localhost:3000)

## 🛠 Tech Stack

* ⚛️ React (Functional components + Hooks)
* 🎨 Tailwind CSS
* 🖥️ No extra libraries!
